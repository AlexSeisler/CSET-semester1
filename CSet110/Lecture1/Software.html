What is Software?
Software: Software is the built-in framework and code that is used to tell the computer what to perform, or display (a set of instructions). This is done through the use of numerous programming languages. These codes are used in two forms, front-end — which includes the look shown in the GUI — and back-end, the actual framework and functionality of the program. This software format is shown in a wide range of areas.
Computer Game: A computer game is very much software. For the game to work, a website or server is needed, but the more specific identification is the storing of data.
Facebook: Facebook is a website formed from many webpages all built using these programming languages that are used to identify something as software, so the answer is yes — Facebook is entirely software.
